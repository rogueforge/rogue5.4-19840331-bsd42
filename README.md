# Rogue 5.4

Reconstructed source for an early version of Rogue 5.4 (Toy, Arnold, Wichman).
Ken Arnold posted this to SourceForge (https://sourceforge.net/p/rogue) in 1999.
It had been brought up to more modern (ANSI C) standards; recreated here is what it
might have looked like in its original BSD 4.2-era form. The special pick-up
and repeat commands are the notable additions. The final 5.4 release had additional
minor changes.
